# Шаг 6: Главная страница

## Цель

Реализовать главную страницу (`/` → `index.php`) с блоками разделов и последними 10 сайтами.

## Маршрут

`/` → `pages/home.php`

## Данные для страницы

### 1. Корневые разделы с подразделами

Запрос:
```sql
SELECT s.*, 
       (SELECT COUNT(*) FROM sites WHERE section_id = s.id AND status = 1) as site_count
FROM sections s
WHERE s.parent_id IS NULL
ORDER BY s.id
```

Подразделы для каждого корневого:
```sql
SELECT * FROM sections WHERE parent_id = :parent_id ORDER BY name
```

Кэш: `sections:tree` (TTL: 1 час)

### 2. Последние 10 опубликованных сайтов

Запрос:
```sql
SELECT st.*, s.name as section_name, s.slug as section_slug
FROM sites st
JOIN sections s ON st.section_id = s.id
WHERE st.status = 1
ORDER BY st.created_at DESC
LIMIT 10
```

Кэш: `sites:recent:10` (TTL: 5 минут)

## Отображение

Без хлебных крошек (главная).

### Блок «Разделы» — сетка из нескольких столбцов

Согласно `main_page_razdels.txt`:
```
[ ДОМ И ИНТЕРЬЕР ]      [ САД И ОГОРОД ]        [ КУЛИНАРИЯ ]
 мебель, декор,          растения,               рецепты,
 текстиль, освещение     инструменты, ландшафт    посуда, напитки

[ ДЕТИ И РАЗВИТИЕ ]     [ ЗДОРОВЬЕ И СПОРТ ]
 игрушки, детск.комната,  фитнес, ПП,
 образование, одежда      медицина, инвентарь
```

Реализация:
- Каждый корневой раздел — карточка Bootstrap
- Название раздела — ссылка на `/section/{slug}`
- Подразделы перечислены через запятую, каждый — ссылка на `/section/{slug}`
- Сетка: `row` + `col-md-4` (3 колонки на десктопе) или `col-md-6` для 2 колонок

### Блок «Последние сайты»

Согласно `main_page_10_sites.txt`:
- Заголовок: «10 последних сайтов в каталоге»
- Нумерованный список
- Каждый элемент: название сайта (ссылка на `/site/{slug}`) + раздел (ссылка на `/section/{slug}`) + дата

### Кнопка «Добавить сайт»

Под блоком разделов, выровнена влево:
```html
<a href="/add" class="btn btn-primary">Добавить сайт</a>
```

## Контрольные точки

- [ ] Главная страница открывается по `/`
- [ ] Разделы отображаются в сетке (2-3 колонки)
- [ ] Каждый раздел — кликабельная ссылка
- [ ] Подразделы перечислены под названием раздела
- [ ] Кнопка «Добавить сайт» отображается
- [ ] Список последних 10 сайтов отображается
- [ ] Каждый сайт кликабелен (ссылка на страницу сайта)
- [ ] Дата отформатирована (ДД.ММ.ГГГГ)
- [ ] Хлебных крошек нет на главной
- [ ] Данные кэшируются в Redis (если доступен)
