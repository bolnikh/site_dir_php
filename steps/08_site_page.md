# Шаг 8: Страница отдельного сайта

## Цель

Реализовать страницу отдельного сайта (`/site/{slug}` → `site.php`) с подробной информацией.

## Маршрут

`/site/{slug}` → `pages/site.php?slug=`

## Алгоритм

1. Получить `slug` из URL
2. Найти сайт по slug (только со статусом `1` — опубликован)
3. Если не найден → 404
4. Получить раздел сайта и всех его родителей для хлебных крошек
5. Отобразить страницу

## Запросы

### Сайт по slug
```sql
SELECT st.*, s.name as section_name, s.slug as section_slug, s.path as section_path
FROM sites st
JOIN sections s ON st.section_id = s.id
WHERE st.slug = :slug AND st.status = 1
```
Кэш: `site:{slug}` (TTL: 30 минут)

### Хлебные крошки (от раздела сайта)

Получаем раздел сайта (`section_id`), затем через `path` строим цепочку:
```
Начало > Дом и интерьер > Мебель > Название сайта
```

(Родители через разбор `path` + CTE, как в шаге 6)

## Отображение

Согласно `site.txt`:

```
─────────────────────────────────────────────────────────
ФИТНЕС ДОМА
─────────────────────────────────────────────────────────
добавлен: 03.06.2026

Онлайн-тренировки для дома без оборудования. Йога, пилатес, кардио,
силовые упражнения. Программы для похудения, набора массы и поддержания
формы. Авторские курсы от профессиональных тренеров. Первое занятие
бесплатно.
─────────────────────────────────────────────────────────
```

### Детали реализации

- Заголовок H1: название сайта
- Горизонтальная линия (или border) для визуального разделения
- Дата: «добавлен: ДД.ММ.ГГГГ», мелкий шрифт, цвет `text-muted`
- Описание: отформатированный HTML (после `clean_html()`)
- Если сайт не опубликован (status != 1) для обычного пользователя → 404

### Хлебные крошки
```
Начало > Название раздела > Название подраздела > Название сайта
         (ссылка)           (ссылка)              (без ссылки)
```

## Контрольные точки

- [ ] Страница открывается по `/site/{slug}`
- [ ] Несуществующий slug → 404
- [ ] Неопубликованный сайт не виден обычным пользователям
- [ ] Хлебные крошки полные (включая всех родителей раздела)
- [ ] Название сайта отображается как H1
- [ ] Дата добавления отформатирована
- [ ] Описание сайта выводится с сохранением форматирования (HTMLPurifier)
- [ ] Кэширование работает
