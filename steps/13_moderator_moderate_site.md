# Шаг 13: Модерация отдельного сайта

## Цель

Реализовать страницу модерации сайта с возможностью одобрить, отклонить или отредактировать сайт, а также отправку email-уведомлений.

## Маршрут

`/moderator/moderate?id={site_id}` → `moderator/moderate.php?id=`

Требуется авторизация модератора.

## Алгоритм

1. Проверить авторизацию
2. Получить `id` сайта из `$_GET['id']`
3. Загрузить сайт (любой статус, не только 0 — чтобы видеть уже обработанные)
4. Обработать действия: `approve`, `reject`, `edit_and_publish`

## Запрос для загрузки сайта

```sql
SELECT st.*, s.name as section_name, s.slug as section_slug,
       s.path as section_path
FROM sites st
JOIN sections s ON st.section_id = s.id
WHERE st.id = :id
```

## Отображение

Согласно `moderator_site.txt`:

### Блок «Данные сайта»
- Раздел (название)
- Название сайта
- URL сайта
- Дата добавления
- Email автора
- Описание (очищенный HTML)

### Блок «Действия модератора»
- Кнопка «✅ ОДОБРИТЬ» — форма с `action=approve`
- Кнопка «❌ ОТКЛОНИТЬ» — форма с `action=reject`
- Обе кнопки с подтверждением (JS `confirm`)

### Блок «Редактирование сайта»
- Селект раздела
- Поле названия
- Текстареа описания (Summernote)
- Кнопка «💾 СОХРАНИТЬ И ОПУБЛИКОВАТЬ»

## Действия

### Одобрение (`action=approve`)
```sql
UPDATE sites SET status = 1, moderated_at = NOW() WHERE id = :id
```
- Инвалидация кэша: `sites:recent:10`, `sites:section:{id}:page:*`, `site:{slug}`
- Если указан email — отправить письмо о принятии (шаблон `site_approved.php`)
- Flash: «Сайт "{name}" одобрен и опубликован!»
- Редирект на `/moderator/list`

### Отклонение (`action=reject`)
```sql
UPDATE sites SET status = 2, moderated_at = NOW() WHERE id = :id
```
- Если указан email — отправить письмо об отклонении (шаблон `site_rejected.php`)
- Flash: «Сайт "{name}" отклонён.»
- Редирект на `/moderator/list`

### Редактирование и публикация (`action=edit_and_publish`)
1. Валидация полей (название, раздел, описание)
2. Обновление:
   ```sql
   UPDATE sites SET section_id = :section_id, name = :name, 
          description = :description, status = 1, moderated_at = NOW()
   WHERE id = :id
   ```
3. Инвалидация кэша
4. Flash: «Сайт сохранён и опубликован!»
5. Редирект на `/moderator/list`

## Отправка писем

Использовать `Mailer` из шага 3:
- `site_approved.php` — тема «Ваш сайт опубликован!»
- `site_rejected.php` — тема «Ваш сайт отклонён»

Текст писем согласно `technical.md`.

## Инвалидация кэша

При любом изменении сайта:
```php
$cache->delete('sites:recent:10');
$cache->deletePattern('sites:section:{$sectionId}:page:*');
$cache->delete('site:' . $site['slug']);
```
