# Шаг 12: Список сайтов на модерации

## Цель

Реализовать страницу со списком сайтов, ожидающих модерации.

## Маршрут

`/moderator/list` → `moderator/list.php`

Требуется авторизация модератора.

## Алгоритм

1. Проверить авторизацию (`require_moderator()`)
2. Получить параметры: `page` (страница), `search` (поиск по названию)
3. Запросить сайты со статусом 0 (на модерации)
4. Отобразить список с пагинацией

## Запросы

### Подсчёт общего количества
```sql
SELECT COUNT(*) FROM sites WHERE status = 0
-- С поиском:
SELECT COUNT(*) FROM sites WHERE status = 0 AND name ILIKE :search
```

### Список сайтов
```sql
SELECT st.*, s.name as section_name, s.slug as section_slug
FROM sites st
JOIN sections s ON st.section_id = s.id
WHERE st.status = 0
  AND (:search IS NULL OR st.name ILIKE '%' || :search || '%')
ORDER BY st.created_at DESC
LIMIT :limit OFFSET :offset
```

## Отображение

Согласно `moderator_list.txt`:

- Заголовок: «🛡️ МОДЕРАЦИЯ — СПИСОК САЙТОВ»
- Поле поиска с кнопкой «Поиск»
- Список сайтов:
  ```
  1.  РемонтПро.ру
      → Дом и интерьер / Ремонт и отделка • 07.06.2026 • ожидает проверки
  ```
- Каждый сайт кликабелен → `/moderator/moderate?id={site_id}`
- Пагинация внизу

### Ссылки для модератора

В хедере (или отдельной панели) добавить ссылки:
- «Список на модерацию» — `/moderator/list`
- «Управление разделами» — `/moderator/sections`
- «Выход» — `/moderator/logout`

## Статусы сайтов

| status | Значение | Отображается |
|---|---|---|
| 0 | На модерации | «ожидает проверки» (жёлтый) |
| 1 | Принят | «опубликован» (зелёный) |
| 2 | Отклонён | «отклонён» (красный) |

На этой странице показываем **только status = 0**. При желании — можно добавить фильтр по статусу для просмотра отклонённых/принятых.

## Контрольные точки

- [ ] Список показывает сайты со статусом 0

- [ ] Клик по сайту ведёт на страницу модерации
- [ ] Пагинация работает
- [ ] В хедере/меню есть ссылки модератора
