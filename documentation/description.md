# Проект - Каталог сайтов

## Стек

PHP (nginx + php-fpm)+ PDO Postgres, css bootstrap. 
Для кеша Redis, отправка почты через smtp.


## Функциональность

1) пользователь может зайти на сайт, авторизация и регистрация не нужна
2) Пользователь может зайти в разделы сайта, страницу отдельного сайта, посетить страницы О нас, Правила, Добавить сайт. 
Он может добавить сайт в форме на странице Добавить сайт. Сайт попадет на модерацию и будет опубликован после модерации.
3) Разделы сайта - это темы / подтемы для группировки сайтов.
4) Список сайтов зависит от раздела сайта и выводится постранично по 20 сайтов (число сайтов настраивается к конфиге).

# Модерация

Поданые сайты попадают на модерацию. После одобрения модератором они публикуются на сайте.
Модератор может отклонить сайт. 
Модератор может перенести сайт в другой раздел.
Модератор может отредатировать название и описание сайта

Модератор авторизуется, по логину и паролю (задаются в конфиге).


## Страницы модерации 

### Логин

На отдельной странице форма логин+пароль. При правильном входе редирект на список сайтов на модерации.

@pages/moderator_login.txt

### Список сайтов на модерацию 

Список сайтов аналогично списку сайтов в разделе, но клик по имени сайта ведет на страницу модерации.

@pages/subrazdel_site_list.txt

### Модерация отдельного сайта

Модерация (заголовок)

УРЛ сайта

Дата добавления

Описание 

Ниже кнопки Одобрить (для одобрения и публикации сайта) и Отклонить (для отклонения).

Еще ниже форма редактирования сайта из полей

раздел ( селект)

Название 

Описание (визуальный редактор)

кнопка Сохранить и опубликовать.

@pages/moderator_site.txt

При одобрении сайта и если указан емейл на него уходит письмо о том что сайт принят.
При отклонении сайта и если указан емейл уходит письмо что сайт был отклонен. 


### CRUD для работы с разделами

Страницы

* Главная - список рубрки в виде дерева с отступами для подразделов, 
кликаем на любой раздел для редактирования или удаления. 
Удалить можно только если в разделе нет сайтов. 
Рядом есть кнопка добавть подраздел в указанный раздел.

Фильтр по имени раздела, если применен фильтр, то выводятся только подходящие разделы с полным путем от корневого раздела

* Страница добавления раздела
* Страница редактирования раздела
